# 決定した追加・改善機能まとめ（シンプル・オープン方針）

方針: 「ログインは会員番号のみ（パスワードやGoogle連携は一切なし）」および「管理者タブを分けず、誰でも練習の作成やExcel出力が可能」という、サークル内の信頼関係に基づいた究極にシンプルで使いやすい運用を目指します。

## 1. 出欠登録の改善（グループ化と一括送信・コメント付与）
* **要件**:
  * 練習を「新歓Hiphop」などの単位（練習名等）で大きなグループカードにまとめる。
  * カードを開くと、そのグループに属する全ての日程がドロップダウン展開される。
  * 複数日程の「出席・欠席・遅刻・早退」および「遅刻・早退の理由（コメント）」を一度にポチポチ一気に入力し、「一括保存」ボタンでまとめて提出・上書き更新ができるようにする。

## 2. プロジェクト単位の出欠一覧表示とExcelエクスポート
* **要件**:
  * 誰でも練習（プロジェクトごと）の画面から、その練習の**対象メンバー全員の出欠状況が一目でわかるマトリックス表**を見ることができる。
  * その表を**ブラウザ側またはバックエンドの無料ライブラリを用いて自動生成し、Excel（.xlsx / .csv）形式などでまとめてダウンロード**できる。
  * 誰でも出欠集計ができるため、特定の幹部の事務的負担が一気に減ります。

## 3. Discordを活用したリアルタイム通知（ジャンル別対応）
* **要件**:
  * 出欠が更新された際に、幹部のDiscordにリアルタイムで通知を飛ばす（完全無料・無制限）。
  * 練習のジャンル情報を見て、Hiphopなら `#hiphop-出欠通知` のように個別のDiscordチャンネルへ通知を自動で振り分ける。
